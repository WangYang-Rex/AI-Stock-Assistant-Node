## Description

基于 [Nest](https://github.com/nestjs/nest) 框架的AI股票助手后端服务，提供智能股票分析和推荐功能。

## 功能特性

- 📊 **股票数据管理**: 股票信息、价格数据、持仓管理
- 🤖 **AI信号分析**: 智能股票信号生成和分析
- 💼 **交易管理**: 交易记录和持仓跟踪
- 📈 **实时报价**: 股票实时价格查询
- 🏥 **健康检查**: 系统状态监控
- 📝 **完整日志系统**: 结构化日志记录，支持多级别、多输出格式

## 日志系统

项目集成了完整的日志系统，支持：

- **多级别日志**: debug、info、warn、error
- **文件输出**: 自动分离错误日志、应用日志、HTTP日志
- **日志轮转**: 自动文件大小控制和数量限制
- **业务日志**: 专门针对股票、交易、数据库操作的日志记录
- **性能监控**: HTTP请求响应时间记录

### 日志文件

- `logs/app.log` - 应用主日志
- `logs/error.log` - 错误日志  
- `logs/http.log` - HTTP请求日志

### 环境变量

- `LOG_LEVEL`: 日志级别 (debug, info, warn, error)，默认: info

## Project setup

```bash

```

## Compile and run the project

```bash

$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Run tests

```bash

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
