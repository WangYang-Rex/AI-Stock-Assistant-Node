# 一、设计原则（先定“性格”）

这不是给你“看热闹”的图，而是 **辅助决策的仪表盘**，核心原则只有 4 条：

1️⃣ **趋势永远在最显眼的位置**
2️⃣ 分时只在「允许交易」时才出现
3️⃣ 风控信号比买入信号更醒目
4️⃣ 所有图形都能直接对应一条“系统规则”

# 三、① 趋势状态总览（核心中的核心）

👉 **这是你第一眼只看 3 秒的地方**

### 推荐展示（卡片式）

```
趋势方向：   ↑ UP
趋势强度：   STRONG
结构状态：   多头排列
策略状态：   ✅ 允许交易
```

### 颜色规则（非常重要）

* UP + STRONG → 🟢 绿色
* UP + MEDIUM → 🟡 黄色
* WEAK / DOWN → 🔴 红色

❗ **只有这里是“判断层”，下面全部是执行**

---

# 四、② 日线 / 60min 趋势图（结构确认）

### 图上只放 5 样东西（再多就是噪音）

* K 线
* MA5 / MA20 / MA60
* EMA20（高亮）
* 趋势区间高低点
* ATR 通道（可选）

### 关键视觉设计

* **EMA20 加粗**
* MA60 灰色、弱化
* 趋势方向用背景淡色带表示（不是线）

👉 你在这里回答的是一句话：

> **“这票现在是不是顺势？”**

---

# 五、③ 分时 / 5min 执行区（只在允许时显示）

⚠️ **重点**：
如果 `趋势层 = 不允许交易`
→ **整个分时区直接灰掉**

---

## 分时图推荐元素（克制！）

### 分时（1min / 分时线）

* 价格线
* VWAP（粗线）
* 成交量柱（底部）

### 5 分钟图

* K 线
* EMA20
* 区间高低框（横盘识别）

---

## 信号可视化方式（不要用“箭头乱飞”）

### 示例：VWAP 回踩

* VWAP 区域 → 半透明色带
* 满足条件 → 小圆点提示
* 鼠标悬停显示规则文字：

```
VWAP 回踩：
✔ 价格接近
✔ 成交量缩
✔ 趋势允许
→ 建议：加仓 10%
```

👉 **不是“叫你买”，而是“告诉你系统怎么看”**

# 六、④ 仓位 & 风控状态（情绪刹车）

这个区域的作用只有一个：
**防止你手贱**

### 仓位条（非常实用）

```
当前仓位： ███████░░░ 70%
系统上限： ██████████ 80%
```

---

### 风控状态灯（醒目但冷静）

* 🟢 正常
* 🟡 波动放大（ATR ↑）
* 🔴 触发止损 / 趋势破坏

当 🔴 出现时：

> 分时图自动锁定
> 所有“加仓按钮”禁用

---

# 七、一个“非常职业”的细节设计

## ❌ 不显示的东西

* 盈亏金额（只显示百分比）
* 日内涨跌排名
* 情绪类指标（什么情绪指数）

👉 **这些东西只会干扰执行**

---

## ✅ 显示但弱化的东西

* 浮盈浮亏（灰字）
* 成交明细（可展开）

---

# 八、前后端数据结构（给你工程直觉）

```ts
interface DashboardState {
  trend: {
    direction: 'UP' | 'DOWN' | 'RANGE'
    strength: 'STRONG' | 'MEDIUM' | 'WEAK'
    allowTrade: boolean
  }
  exec: {
    vwapSignal: boolean
    ema20Pullback: boolean
  }
  position: {
    current: number
    max: number
  }
  risk: {
    level: 'GREEN' | 'YELLOW' | 'RED'
  }
}
```

前端**只渲染状态，不做判断**
👉 所有判断都来自你的模型，这是专业系统的分工。
