推荐目录接口

```
src/
├── core/                # 全局核心：拦截器、过滤器、守卫、基础配置
├── common/              # 公共资源：装饰器、DTO基类、通用Service、Constants
├── entities/            # (可选) 集中管理 TypeORM 实体，或保持在各模块内
├── modules/             # 业务逻辑集中地
│   ├── market/          # 市场数据领域 (聚合相关模块)
│   │   ├── kline/       # K线子模块
│   │   ├── quotes/      # 报价子模块
│   │   └── trends/      # 分时子模块
│   ├── info/            # 基础信息领域
│   │   ├── stock/       # 股票基础信息
│   │   └── tags/        # 标签/分类信息
│   └── user/            # 用户相关领域 (后期扩展)
├── infrastructure/      # 外部系统集成：DB配置、第三方SDK封装、Redis
└── main.ts              # 入口文件
```
