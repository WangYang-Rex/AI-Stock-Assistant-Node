### 网格交易策略模拟

#### 网格交易参数设置
- 股票 stock = "588080"
- 昨收价格: previousClosePrice = 1.43
- 今日快照数据 src/lib/stockdemo/quotes_time_price.json 读取
- 网格交易幅度 grid_gap = 0.02
- 持仓数量 holdingQuantity = 1000000
- 持仓价格 holdingCost = 1.347

#### 交易策略
操作初始价格=昨收价格
设定下一买入价格=操作初始价格*(1-grid_gap)，到达该价格则买入，手续费为5元
设定下一卖出价格=操作初始价格*(1+grid_gap)*3000股，到达该价格则卖出，手续费为5元
当卖出或者买入后，更新操作初始价格=卖出价/买入价 重复上述步骤
